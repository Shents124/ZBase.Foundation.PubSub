# ZBase.Foundation.PubSub

## OpenUPM dependencies

```bash
openupm add org.nuget.system.runtime.compilerservices.unsafe
openupm add com.cysharp.unitask
openupm add com.zbase.collections.pooled
```

### Install via Package Manager
1. Open the Package Manager from Window > Package Manager
2. `+` button > Add package from git URL
3. Enter the following to install

**Install ZBase.Foundation before installing ZBase.Foundation.PubSub**

```
https://github.com/Shents124/ZBase.Foundation.git?path=/Packages/ZBase.Foundation
```
